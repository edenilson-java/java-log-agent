# 01 - Estrutura Inicial e Dependências

## Objetivo

Inicializar o projeto `java-log-agent` com a estrutura de diretórios, configuração do Git, variáveis de ambiente e dependências fixadas.

## Requisitos

- Usar Python 3.12.10.
- Criar uma virtualenv local em `.venv`.
- Instalar somente as dependências declaradas em `requirements.txt`.
- Manter `OPENAI_API_KEY` vazia no arquivo `.env.example`.
- Copiar somente os três logs iniciais da implementação de referência.
- Não criar código do agente neste bloco.
- Versionar e enviar o resultado para a branch `main`.

## Validações

- Criação correta da virtualenv.
- Instalação completa das dependências.
- Execução de `pip check` sem dependências quebradas.
- Commit e push para o repositório remoto.
